and Type Destination), one Entry for the input number by user, two buttons (Invert Convertion and Convert)
and one label result.

Temperature. The picker Last Conversion save until five Conversion, when is selected one of yours save 
conversions, the program change the elements for be same the configuration her. The program accepted double
and integer number and the label result can have text with integer or double representation in string.
